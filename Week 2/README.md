# Week 2 – Tesla Deliveries Forecasting

## Overview

This project analyzes Tesla deliveries data (2015–2025) using Exploratory Data Analysis (EDA), feature engineering, machine learning, and time series forecasting techniques.

## Tasks Performed

* Loaded and explored the dataset.
* Checked missing values and duplicates.
* Created 5 EDA visualizations.
* Applied Label Encoding on categorical features.
* Created Lag and Rolling Mean features.
* Trained and evaluated a Linear Regression model.
* Performed 5-Fold Cross Validation.
* Tuned a Random Forest Regressor using GridSearchCV.
* Conducted ADF stationarity testing.
* Generated a forecast comparison table with error percentages.
* Documented all findings with markdown explanations.

## Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn, Statsmodels
